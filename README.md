# Inventory Control System

Inventory control system developed to practice Full Stack development concepts using JavaScript, Node.js, Express, and MySQL.

## Features

* Product registration
* Product listing
* Integration with MySQL database
* REST API with Express
* Responsive interface

## Technologies

* HTML
* CSS
* JavaScript
* Node.js
* Express
* MySQL
* Git & GitHub

## Future Improvements

* Product editing
* Product deletion
* Stock movement history
* Dashboard with indicators
* User authentication
